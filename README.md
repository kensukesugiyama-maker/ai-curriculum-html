# AIの使い方カリキュラム（静的HTML）

「わからないからできない」を減らすための対話・マネジメント型のAI活用法を1ページにまとめたものです。

## 共有リンク（GitHub Pages）

リポジトリをPushしたあと、GitHub の Settings → Pages で **main ブランチ / root** を選ぶと、次のURLで閲覧できます。

`https://kensukesugiyama-maker.github.io/ai-curriculum-html/`

（初回は反映に1〜2分かかることがあります。）

## ローカルで開く

- `index.html` または `ai-no-tsukaikata-curriculum.html` をブラウザで開く。

## 内容の編集

- どちらか一方を正とし、もう一方はコピーで揃えるか、片方に統一することを推奨します。
